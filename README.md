# Gas Detection Project Landing Page

This project is a React-based landing page for the Gas Leakage Detection ML project. It uses Tailwind CSS for styling and Framer Motion for animations.

## Getting Started

1.  **Install Dependencies**:
    Open a terminal in this directory and run:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    Run the following command to start the local development server:
    ```bash
    npm run dev
    ```
    Typically, this will be available at `http://localhost:5173`.

## Project Structure

-   `src/components/`: Contains all UI components (Hero, Dataset, Models, etc.)
-   `src/App.jsx`: Main application component assembling the landing page.
-   `src/index.css`: Global styles and Tailwind directives.

## Tech Stack

-   React
-   Tailwind CSS
-   Framer Motion
-   Lucide React (Icons)
-   Vite (Build Tool)
